##  Description

**Simulateur de Trafic** est un package Python permettant de simuler le comportement d’un trafic routier simple.  
Il permet de modéliser des **voitures**, **routes**, et **feux de circulation**, et d’observer leur interaction dans un environnement simulé.

Ce projet a été développé dans le cadre d’un **TP Python**.
